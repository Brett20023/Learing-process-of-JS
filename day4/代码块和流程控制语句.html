代码块 我们的程序是一条条语句构成的 语句是有上自下按顺序一条条执行的 在JS中可以使用{}来为语句进行分组 同一个{}中的语句称为一组语句，要么都执行要么都不执行 一个{}中的语句称为代码块 在代码块后面不用再编写； JS代码块，只具有分组功能，没有其他用处 代码块内部的内容在外面是完全可见的 流程控制语句 -通过流程控制语句可以控制程序流程，使程序可以根据一定条件执行 -语句的分类 1.条件判断语句 使用条件判断语句可以在执行某个语句之前进行判断，如果条件成立才 会执行语句，如果条件不成立则不会执行语句 if语句 -语法一 if{条件表达式} 语句： if语句在执行时，会先对条件表达式进行求值判断，如果条件表达式 的值为true，则执行if语句，否则不会执行 -语法二 不加代码块直接写if的话if就只能控制其后的一条代码，如果要if控制 多条代码的执行要加上代码块 -语法三 if{}else{}语句，如果为true执行if的语句否则执行else的语句 else if可以添加多个判断，知道找到一个结果为true的条件或所 有条件都不满足为止 2.条件分支语句 条件分支语句，也叫switch语句 语法： switch(条件表达式){ case XXXX：XXXX } 在执行时会依次将case后的表达式的值和switch后的条件表达式进行全等比较 如果比较结果为true则会从当前case处开始执行代码（当前case后的代码都会执行 如果结果为false则会继续向下比较 如果想在中途退出switch语句则可以在代码结尾处加上break 如果所有结果都为false则只执行default后的语句 3.循环语句 while(条件表达式){ 语句..... } -while语句先对条件表达式进行判断，如果为true则进行一次计算， 然后返回再进行条件表达式中的判断，直到结果为false为止 -如果将条件表达式中的值写死为true则为死循环，该循环不会停止， 直到关闭浏览器，可以使用break终止死循环 -循环有三个要素 1.初始化变量 2.有一个条件表达式 3.定义一个更新表达式，更新初始变量 -do...while循环 do{ }while() 一上来就会先进行一次循环然后再进行判断,可以保证至少执行一次循环 for(初始化表达式；条件表达式；更新表达式){ 语句.... } 在for循环中有专门的位置来放置三个表达式 for循环执行的流程： 1.执行初始表达式，初始化变量 2.执行条件表达式，判断是否执行循环 3.更新表达式，执行完更新表达式继续重复第二部 prompt(); prompt()可以弹出一个提示框，该提示框中会带有一个文本框，用户可在文本框中输入一段内容， 该函数需要一个字符串作为参数，该字符串姜维作为提示框的文字 用户输入的内容将会作为函数的返回值返回，可定义一个变量来接收该内容 prompt()函数的返回值是string类型 var XXX = prompt('dfasfdasdf'); if(xxx>100){ 1111 } 可以使用sort()函数来对数组中的对象按字母排序， 如果是数字数组则要使用sort(sortNumber)进行排序 使用方法例：A[12,12,455,456] A.sort(sortNumber)
